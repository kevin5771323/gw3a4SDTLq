# 基于Vue3的后台管理系统 java1119

## 项目概述

本项目是基于Vue3的后台管理系统，支持普通用户与管理员用户的登录切换。系统提供丰富的功能模块，用户可根据需求自行选择启用或关闭，包括但不限于富文本编辑器、网站色调配置、bug日志反馈等。

## 技术栈

- **构建工具：** Vite
- **前端框架：** Vue
- **状态管理：** Pinia
- **路由管理：** Vue-router
- **编程语言：** Typescript
- **网络请求：** Axios.js
- **数据模拟：** Mock.js
- **UI组件库：** Element-plus

## 功能模块

- 用户登录切换
- 系统主题更改
- 富文本编辑器
- 网站色调配置
- Bug日志反馈

## 系统角色

- 普通用户
- 管理员用户

## 运行环境

- Node.js 16.2.2版本

## 部署步骤

1. 克隆或下载项目代码。
2. 在项目根目录下打开终端，执行 `npm i` 安装依赖。
3. 依赖安装完成后，运行 `npm run dev` 启动开发服务器。

## 核心亮点

- **灵活性：** 系统功能模块可根据需求进行自由取舍。
- **易于上手：** 使用成熟的UI组件库，降低开发门槛。
- **数据模拟：** Mock.js的引入，方便前端独立开发。
